OBJ examples

Note: there are file references in the examples that assume that
your current directory is lib, large, or proofs.

in directory lib:
*** small parameterized modules
*** reductions for set.obj are in set-exs.obj, etc.
array.obj      -- arrays
bubbles.obj    -- bubble sort
iter.obj       -- iterate a function over a list
lexl.obj       -- lexical ordering on a list
list.obj       -- lists
list2.obj      -- alternate version of lists (used by some examples)
map.obj        -- mapping a function over a list
propc.obj      -- propositional logic decision procedure
set.obj        -- sets
set2.obj       -- a less abstract and more efficient version of sets
sorting.obj    -- sorting a list
theory.obj     -- some sample theories
views.obj      -- some sample views

in directory large:
sieve.obj       -- list of primes using lazy evaluation
unify.obj       -- unification of terms
fun.obj         -- semantics of simple language
cat.obj         -- category theory
combinators.obj -- Combinatory algebra (actually a pretty small example)

in directory proofs:
*** proofs using OBJ3
2not.obj       -- circuit with invertors in series
assoc-add.obj  -- associativity of addition
assoc-add2.obj -- newer variant of the above
comm-add.obj   -- commutativity of addition
comm-add2.obj  -- newer
fadd.obj       -- full adder
map.obj        -- parameterized module MAP
mmm.obj        -- Fermat's little theorem for p=3
mmm2.obj       -- newer
propc.obj      -- propositional logic decision procedure
propc-tst.obj  -- sample reductions for the above
ripple.obj     -- ripple carry adder
sum.obj        -- proof of formula for sum of natural numbers up to n
sum2.obj       -- newer
xor.obj        -- exclusive-or circuit
groupl.obj     -- standard group theory example
groupla.obj    -- variation on the above
inj.obj        -- an injective function with a right inverse is an isomorphism

in directory source/trim/examples
*** a number of simple examples of the use of the TRIM system.
